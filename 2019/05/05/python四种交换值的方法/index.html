python四种交换值的方法

  view

一

第一种方法是运用媒介的方法，找到一个中间值来实现两个数的交换。

a=3
b=4
c=0
c=a
a=b
b=c

二

第二种是python独有的一种方法。

a=3
b=4
a,b=4,3

三

第三种是一种简单的算法。

a = a + b
b = a - b
a = a - b

四

第四种是通过异或来实现的。

a = a^b
b = a^b # b = (a^b)^b = a
a = a^b # a = (a^b)^a = b
