mysql的基本命令

  • MySQL
  view

前言

使用命令行可以快速的实现增删改查，可以对数据库进行修改，可以对数据库中的表进行修改，熟练掌握这些命令，可以在开发中起到事半功倍的作用。

数据库操作

  • 创建数据库
    create database 数据库名 charset=utf8;

  • 删除数据库
    drop database 数据库名;

  • 切换数据库
    use 数据库名;

  • 查看当前选择的数据库
    select database();

表操作

  • 查看当前数据库中所有表
    show tables;

  • 创建表
    create table 表名(列1,列2,...);

    唯一标识的要求：id
    类型：int unsigned
    约束1：not null
    约束2：primary key
    约束3：auto_increment

例：
create table stu( id int not null primary key auto_increment, name varchar(10) not null, gender bit default 1, birthday datetime, isDelete bit default 0);
结果如下图：

  • 查看当前表
    show tables;

  • 查看表结构
    desc 表名;

  • 修改表
    alter table 表名 add|modify|drop 列名 类型 约束;

  • 删除表
    drop table 表名;

  • 更改表名称
    rename table 原表名 to 新表名;

  • 查看表的创建语句
    show create table 表名;

数据操作

  • 添加数据
    insert into 表名(列名，列名) values(值，值);

  • 修改数据
    update 表名 set 列1=值1,... where ...;

  • 删除数据
    delete from 表名 where ...;

  • 查看所有数据
    select * from 表名

数据库的备份与恢复

备份与恢复不需要连接到数据库，只需要在cmd下就可以对数据库进行。

  • 备份
    mysqldump –uroot –p 数据库名 > 目录/备份文件.sql;

  • 恢复
    mysql -uroot –p 数据库名 < 目录/备份文件.sql;

END！
