python的交互式编程与脚本式编程

  view

交互式编程

交互式编程不需要创建脚本文件，是通过 Python 解释器的交互模式进来编写代码。

linux上你只需要在命令行中输入 Python 命令即可启动交互式编程。
我用的是Windows系统，Windows上需要到python官网上下载编译器，然后在PATH变量中添加python，python建议大家都用python3，因为以后可能python2就不会再更新了，所以希望大家用python3。在设置好这些之后你就可以在cmd.exe中输入python，可以得到以下界面：

出现这个页面，说明你已经进入了python的交互模式，在这里面你可以看到自己所使用python的版本，也可以进行简单的四则运算，python的基本语法，输入输出。
例如输出简单的hello world：你可以输入print (‘hello world’)

就可以得到想要的结果。
