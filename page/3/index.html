人生苦短，我用python

Python是什么

python 是一种面向对象、解释型计算机程序设计语言，由Guido van Rossum于1989年圣诞节为打发无聊时间，而开发的一个新的脚本解释程序，可以感觉下什么叫牛人，是ABC语言的一种继承，至于为什么选中Python作为语言名字，是因为他是一叫Monty Python的喜剧团体的爱好者，第一个公开发行版发行于1991年。
他的设计哲学是优雅，明确，简单，完全的面向对象。函数，模块，数字，字符串都是对象，不像Java中还有基本类型，在Python中一切皆对象。

为什么用Python

这个问题往往是入门者第一个问题，对此我在一本书找到了这样的解答：
软件质量
开发效率
程序的可移植性
众多标准库支持
组件集成
享受乐趣
我自己感觉python还是比较简单的，相对于c语言和java语言书写起来都相对比较简单，但也不能说python就能完胜于其他语言，只能说各有千秋，对于python语言，我总结几点就是：
代码量小
维护成本低
编程效率高
同一样问题，用不同的语言解决，代码量差距太多了，一般情况下python是java的1/5，所以说人数苦短，我用python，多留点时间学习其他的，就算不学习，多点时间娱乐也好啊，哈哈。

Python是脚本语言吗

其实它是一种多用户语言，至于为什么大家的第一感觉是脚本语言，我是感觉人们用它直接写一个文件，不需要什么编译，跟脚本似得，直接运行的就行了。所以说我也很难有一个确定答案。
脚本：可以写一些辅助自己开的脚本，就比如，Android开发，会涉及到一常用的命令，但是如果是在windows用bat写，这样弄到mac就没法运行的，所以可以用python写。

Python版Hello World

我感觉学任何一门语言都有一个hello world的过程，python也是如此，因为通过最简单的你会学习到该语言的一个最基本的程序框架和运行过程。
python中输出helloworld我感觉是最简单的了，因为它值需要一行。
print （'hello world'）
顺利的话你会看到hello world的输出，是不是感觉好简单，对，你没看错，就这么简单，现在你可以说你是一个python程序员了。

提示

在学习过程中，大家也要做好python2和python3的区别，有些时候可能你的代码没有错，是用错了python版本的编译器。

end！

githu+hexo搭建个人博客

总结一下我自己搭建博客的过程，感觉自己还是挺慢的，实验室让我们自己搭建博客的时候我感觉很难弄，当弄好了之后我感觉也就那样吧，我是看了许多别人的博客和视频，自己感觉他们弄的有一些东西说的不是很详细，或许我的这一篇写的也不是很完整，希望有大佬可以批评指教。

前言

首先说一下为什么选择github+hexo搭建博客吧，我感觉最直接的就是不用一分钱，对于这种免费的事情，我认为大家都乐意去做，哈哈。另外还有其他的好处，比如

全是静态文件，访问速度快；
可以随意绑定自己的域名，不仔细看的话根本看不出来你的网站是基于github的；
数据绝对安全，基于github的版本管理，想恢复到哪个历史版本都行；
博客内容可以轻松打包、转移、发布到其它平台；

准备

我是用的Windows系统搭建的，其他系统请另寻捷径，其实也都没有多大差距，但有时候一点小问题都能让你不想继续下去，所以还是找相应系统的教程比较好。在开始一切之前，你必须已经：

有一个github账号，没有的话去注册一个；
安装了node.js、npm，并了解相关基础知识；
安装了git for windows（或者其它git客户端）

My first blog

这是我的第一篇博客，实在是不想第一篇就写技术类的文章，主要是我也没有技术，哈哈。写一些什么好呢？现在上大一，就先写一些我的感想吧，作为我的起点。

首先，我是小菜鸟一枚，以下的观点可能不正确，欢迎批评指正。

大一上学期时间就这样的过去了，经过了半年的时间，我也有了很深的体悟：大学，不是我们想的那么简单!上学期，感觉时间过的好快，自己刚退伍回来，已经两年没有了学习，在部队，天天都是一些不费脑子的活动，再加上自己对计算机也不是很理解，开学一段时间，不知道自己天天学的是啥，很迷茫。这一学期下来，基本上没有听过课，上课都是在学习自己喜欢的计算机知识或者是看一些其他的。

对于程序设计，以前的我总是自我感觉良好，觉得老师教的都会，留的实验报告都能做出来，然后我就飘飘然，现在感觉自己的目光太短浅了，程序设计并不是自己想的那么简单，自己才学到的是简单的语法和结构，深入的算法都一无所知。

对于信息安全，刚开学时，自己加入了信息安全实验室，那时感觉自己对这方面挺热爱，感兴趣，当学了一段时间之后，发现都没有那么简单，好多时候都是做了一晚上一道题都做不出来，但一学期过去了，虽然我还是感觉技术没有增加多少，但我还是挺喜欢信息安全的，并相信自己能一直坚持学下去。

现在的我，有时候有些迷茫，特别是在有很多题做不出来，或者是怎么也做不对的时候。但我从未放弃过，自己选择的路，跪着也要走完！在经过着一学期之后，我在学习上也有了许多方法，不再像刚入学的时候什么都不知道了，也慢慢的懂得了自己要学的方向。

其实无论什么事情，坚持最重要，若害怕困难，还能搞什么事情？？？做一个不忘初心的人。
